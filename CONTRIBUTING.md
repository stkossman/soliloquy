Thank you for your interest in contributing to Soliloquy! 
Since this is a local-first project, please ensure you test changes with a clean IndexedDB state if modifying the schema.

### Pull Request Process
1. Fork the repo and create your branch from `main`.
2. Ensure you use **Bun** for package management.
3. Run `bun run check` and `bun run lint` before committing.
4. Follow the Conventional Commits naming style.
